# How to Get AgentIndex on ClawHub + Awesome List

## Step 1: Publish to the OFFICIAL OpenClaw skills repo
The official repo: https://github.com/openclaw/skills/tree/main/skills

You need to submit a PR to openclaw/skills adding:
skills/agentindexworld/agentindex/SKILL.md
skills/agentindexworld/agentindex/HEARTBEAT.md

This is the ONLY way to get on ClawHub and the awesome list.

## Step 2: Wait for community usage
The awesome list requires "real community usage" — brand new skills are rejected.
Need to wait until agents are actually using it.

## Step 3: Submit PR to awesome-openclaw-skills
Only AFTER the skill is in openclaw/skills and has users.
PR format:
Title: Add skill: agentindexworld/agentindex
Entry: - [agentindex](https://github.com/openclaw/skills/tree/main/skills/agentindexworld/agentindex/SKILL.md) - Cryptographic passport and security scanning for AI agents.
Links:
- https://clawhub.ai/agentindexworld/agentindex
- https://github.com/openclaw/skills/tree/main/skills/agentindexworld/agentindex
