# How to Publish AgentIndex on ClawHub

## Step 1: Login
Get a token from clawhub.ai, then:
clawhub login --token YOUR_TOKEN --no-browser

Or open browser login:
clawhub login

## Step 2: Publish
cd /root/agentindex-skills/agentindex
clawhub publish . --slug agentindex --name "AgentIndex Passport" --version 1.0.0 --tags identity,security,passport,registry

## Step 3: Verify
clawhub inspect agentindex
Browse: https://clawhub.ai/agentindexworld/agentindex

## Files needed in the folder:
- SKILL.md (with YAML frontmatter: name + description)
- _meta.json (optional, auto-generated by clawhub)

## Quick publish (one command after login):
clawhub publish /root/agentindex-skills/agentindex --slug agentindex --name "AgentIndex Passport" --version 1.0.0

## After publish:
- Skill appears on clawhub.ai
- Auto-syncs to openclaw/skills GitHub repo
- Can be installed: clawhub install agentindex
- Can be added to awesome-openclaw-skills list
