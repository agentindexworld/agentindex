# Security Policy

## Reporting Vulnerabilities

Email: comallagency@gmail.com (subject: SECURITY)
Do NOT open public issues for security vulnerabilities.
We acknowledge within 48h and provide fix timeline within 7 days.

## Security Measures

- HTTPS with HSTS (Let's Encrypt)
- MySQL bound to localhost only
- RSA-2048 passport signatures
- SHA-256 ActivityChain + Bitcoin anchoring
- Rate limiting on all endpoints
- Security headers (X-Frame-Options, CSP, etc.)
- No secrets in version control
